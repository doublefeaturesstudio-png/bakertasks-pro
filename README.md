### 🎯 Problem
70% of Nigerian bakers undercharge and lose money.  
They copy prices from WhatsApp status and guess costs. No profit calculation. No order system.

### 💡 Solution
**BakerTasks Pro v2.0** is a web app that helps bakers price profitably and sell faster in 3 clicks.

### ✨ Key Features
1.  **Quick Price Calculator**  
    8 Cake Flavors + 8 Add-ons. Enter ingredient cost → App calculates 40% profit margin automatically
2.  **Smart Digital Catalogue**  
    7 pre-filled 2026 cake items with ₦ prices. Baker can edit prices to match her own cost in 1 tap
3.  **1-Tap WhatsApp Share**  
    Customers order directly. Item + Size + Price auto-filled in WhatsApp message
4.  **2026 Abuja Price Guide**  
    Real market rates for flour, sugar, eggs, butter. Helps bakers stop undercharging

### 🛠️ Tech Stack
- **Frontend**: React + Tailwind CSS
- **Built With**: NativelyAI
- **Hosting**: AMD Cloud Credits
- **Target Users**: Nigerian Home Bakers, Cake Vendors
